=== Attach Gallery Posts ===
Contributors: dkukral
Tags: gallery, media, posts
Requires at least: 2.9.1
Tested up to: 4.0
Stable tag: 1.6

Plugin for WordPress that allows you to attach a post with a gallery to another post.

== Description ==

Used to add galleries to posts in a Wordpress site.

Copy "attach-gallery-post" folder to your WordPress Plugins Directory. Activate plugin via WordPress settings.
